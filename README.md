# 👨‍💻 Vu Trung Kien - Software Engineer Portfolio

Welcome to the source code of my personal portfolio website! This project serves as a dynamic, interactive showcase of my experience, education, and technical skills as a Software Engineer.

![Portfolio Preview](./src/assests/images/seo-img.png)

## ✨ Key Features

- **Modern Tech Stack**: Built with React 18 and powered by **Vite** for blazing-fast HMR and optimized builds.
- **Multi-language Support (i18n)**: Seamlessly toggle between English (EN) and Japanese (JP) using local storage.
- **Dynamic Theming**: Full Dark/Light mode support tailored to the user's system preferences.
- **Responsive Design**: Mobile-first CSS styling ensuring perfect display across all devices (including a custom CSS-only mobile hamburger menu).
- **Interactive UI**: Fluid animations powered by `react-awesome-reveal`.
- **Project Deep-dives**: Detailed project modals featuring ER diagrams, functional maps, and API flow breakdowns.

## 🚀 Tech Stack

- **Frontend**: React.js (v18), Vite, React Router v7, HTML, CSS
- **Styling**: Vanilla CSS, BaseUI, Styled Components
- **Animations**: React Awesome Reveal
- **Backend (Experience)**: Node.js, NestJS, Express.js
- **Cloud & DevOps**: AWS, Docker, Microservices, CI/CD

## 🛠️ How to run locally

1. **Clone the repository:**

   ```bash
   git clone https://github.com/KieenVux/MyProfolio.git
   cd MyProfolio
   ```

2. **Install dependencies:**
   _(Note: This project uses legacy-peer-deps to maintain compatibility with specific UI libraries)_

   ```bash
   npm install --legacy-peer-deps
   ```

3. **Start the development server:**

   ```bash
   npm start
   ```

   _The Vite dev server will start and expose the app on your local network (e.g., `http://localhost:3000`)._

4. **Build for production:**
   ```bash
   npm run build
   ```
