# Sales Network

A Smart HRM & Payroll Management system built for scaling Sales organizations. This backend-heavy project utilizes a modern Microservices architecture to handle strict Manager Approval workflows, KPI Bonus/Penalty processing, and asynchronous mass Payroll transfers via 9Pay.
